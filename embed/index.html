Update

  • Update

Upload a pixelix firmware for software update or a filesystem binary for updating the filesystem.

A filesystem update might cause a loss of all configuration data on the filesystem. Make sure you have a backup of your configuration before updating it. Backups can be done via the webinterface of Pixelix.
A filesystem update will disable the HomeAssistant MQTT discovery feature. After the update, you have to re-enable it manually. This prevents that the welcome plugin will be discovered, after a filesystem update.
